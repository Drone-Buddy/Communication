# Communication

## Install As A Background Service

```
git clone https://github.com/Drone-Buddy/Communication.git
cd Communication
./install_dronebuddy_software.sh
```

## Run Normally

```
python RaspberryPiWebsocketServer.py
```

It require pip packages:
* websocket_client
* json
* pyserial
* datetime
* thread
* time
